# Project Memory Index

Acest folder este compatibil cu Obsidian si pastreaza deciziile stabile ale proiectului. Notele nu inlocuiesc template-ul oficial Word pentru lucrare; ele sunt memoria tehnica si metodologica a proiectului.

## Note principale

- [[01-title-and-requirements]] - titlul fixat, template-ul 2026 si cerintele administrative.
- [[02-dataset-and-eol-thresholds]] - datasetul ales, provenienta si pragurile EOL/SOH.
- [[03-experiment-design]] - scenarii, tinte, split-uri, modele si reguli de evaluare.
- [[04-results-and-interpretation]] - rezultate curente si interpretare onesta.
- [[05-final-project-overview]] - cum va arata proiectul final si demo-ul Streamlit.
- [[06-rul-soh-debugging-notes]] - erori gasite, reparatii si interpretarea noilor rezultate.

## Directia finala

Lucrarea combina doua formulari complementare:

1. predictie directa RUL, exprimata in cicluri ramase;
2. predictie SOH/capacitate, urmata de derivarea RUL pana la un prag EOL.

Aceasta structura ramane aliniata cu titlul oficial deoarece RUL este obiectivul final, iar SOH este indicatorul fizic prin care estimarea devine mai stabila si mai usor de explicat.
