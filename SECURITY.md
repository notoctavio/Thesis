# Security Policy

## Supported versions
This repository is an academic thesis project. Security fixes are handled on the active branch used for current thesis work.

## Reporting a vulnerability
Please do not open public issues for sensitive security findings.

Instead, contact the maintainer directly and include:
- A clear description of the issue
- Reproduction steps
- Potential impact
- Suggested mitigation (if available)

I will acknowledge reports and evaluate fixes as quickly as possible.
